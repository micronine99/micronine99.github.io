测试

1、安装hugo

2、检查安装 hugo version

3、创建blog hugo new site myblog

![img](D:\New app\有道云笔记\pty63506350@163.com\ca79736d5de743caac6b65c0c468c691\clipboard.png)

4、下载主题

在themes文件夹下： git clone https://github.com/syui/hugo-theme-air

ls 打开之后可以看到

5、在本地启动blog

hugo server -t hugo-theme-air –buildDrafts

6、新建文章

hugo new post/blog.md 【需要在主题的content里new】

7、第五步，发布

8、部署到github

![img](D:\New app\有道云笔记\pty63506350@163.com\f06c942021ee4304a85cbcd27f4e39fa\clipboard.png)

根目录下:hugo –theme=hugo-theme-air –baseUrl="https://micronine99.github.io/” –buildDrafts

![img](D:\New app\有道云笔记\pty63506350@163.com\23afcd279838416fb2937e2f80865fd5\clipboard.png)

将public文件夹传到远端：

在public里面：

git init -> git add * -> git commit -m “xxxx”

连接远端：git remote add origin1 https://github.com/micronine99/micronine99.github.io.git

推出：git push -u origin1 master

至此就可以用micronine99.github.io访问
